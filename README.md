
# Codeial-Web-Application

A small sample web application where you can add a post, add comments to the post and also reply to comments, user can also delete unusual comments, You can create profiles Authenticated checks are everywhere in the application and data is stored permanently in the server.


## Installation


```
    Install Node.js: Ensure that Node.js is installed on  system.  Download it from the official website: https://nodejs.org/

Clone the repository: Would need to clone your project's repository from GitHub using git clone or by downloading the project as a ZIP file.

Install project dependencies: In the project directory, should run the following command to install the required dependencies specified in the package.json file: npm install

Install MongoDB: MongoDB should be installed and running on the system. Download and install MongoDB Community Edition from the official website: https://www.mongodb.com/try/download/community

Configure MongoDB connection: The person would need to modify the config/mongoose.js file to specify the connection details for their MongoDB instance. This includes the host, port, database name, and authentication credentials if applicable.

Run the project: After completing the above steps,  can start  project by running the following command in the project directory:  node index.js


Alternatively, they can use a tool like nodemon to automatically restart the server on code changes during development: npm install -g nodemon
nodemon index.js


This will start the server, and should see a message indicating that the server is listening on a specific port (e.g., Listening on port 8000).

Access the project: Once the server is running, can access the project by opening a web browser and navigating to http://localhost:8000 (or the appropriate port specified in project).


npm start to start the server
## Features

- User can make multiple profiles in this application  
- User can add a post 
- User can reply to a post
- User can also reply to someone's reply
- Can delete a comment, post, reply on post
- User can update his profiles
- User can create multiple friends
- Sign-UP 
- Sign-in 
- Logout
